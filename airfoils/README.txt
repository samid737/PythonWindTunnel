The dat files contain airfoil coordinates in SELIG and LEIDNICER format.the program uses SELIG. Many other airfoil coordinates can be found at:

http://www.windandwet.com/windturbine/airfoil_plotter/index.php

PLEASE NOTE THAT YOU MUST REMOVE THE FIRST LINE CONTAINING THE NAME OF THE AIRFOIL FOR THE PROGRAM TO WORK

The correct parsing format is:

somecoordinate     anothercoordinate

so five spaces in between. a lot of airfoils are not in the correct format.    

use this tool to remove any leading spaces:

http://nimbletext.com/Live/1970117955/

use this tool to replace any incorrect spacing:

http://nimbletext.com/Live/-308576193/


you can save the files to any filetype, .dat, .txt ...
